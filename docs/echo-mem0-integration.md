## 二、记忆分层策略

### 第一层：Mem0 知识库（所有经验和数据）

**用途**: 存储所有重要的经验、数据、策略

**存储内容**:
1. **公众号运营经验**
   - 高转化率的选题方向
   - 有效的文章结构和风格
   - 读者反馈和偏好

2. **小红书获客数据**
   - 高质量笔记的特征
   - 有效/无效的回复话术
   - 关键词转化率数据

3. **审批偏好学习**
   - 人工批准的案例特征
   - 人工拒绝的案例特征
   - 修改建议的模式

4. **策略优化记录**
   - 采集时间段效果分析
   - 平台优先级调整历史
   - A/B 测试结果

### 第二层：飞书多维表格（结构化数据）

**用途**: 记录可量化、需要协作的数据

- **内容日历表**: 发布时间、文章标题、阅读量、转化客户数
- **小红书笔记库**: 笔记链接、质量评分、是否已回复
- **意向客户池**: 评论用户、需求类型、审批状态

## 三、使用示例

### 1. 存储公众号经验

```javascript
memory_store({
  content: "[公众号] 平台政策解读类文章 - 转化率最高\n时间：2026-03-13\n效果：转化率 15%，阅读量 2000+\n来源：《Cdiscount 2026 最新招商政策解读》"
});
```

### 2. 存储小红书话术

```javascript
memory_store({
  content: "[小红书话术] 店铺咨询类回复模板 - 先提供价值再引流\n时间：2026-03-13\n效果：审批通过率 80%，私信回复率 60%\n模板：您好！关于XX平台，我们整理了一份最新的入驻指南，可以免费分享给您。如果需要详细咨询，欢迎加微信：XXX"
});
```

### 3. 存储关键词效果

```javascript
memory_store({
  content: "[关键词效果] cdiscount - 高转化关键词\n时间：2026-03-13\n效果：采集 50 条，意向客户 18 人，转化率 36%\n建议：优先级调整为"高"，增加采集频率"
});
```

### 4. 搜索记忆

```javascript
// 搜索关键词效果
memory_search({
  query: "cdiscount 关键词效果"
});

// 搜索公众号经验
memory_search({
  query: "公众号 平台政策 转化率"
});

// 搜索审批案例
memory_search({
  query: "审批案例 店铺咨询"
});
```

## 四、存储格式规范

### 统一格式

```
[类型] 主题 - 核心内容
时间：YYYY-MM-DD
效果：转化率/阅读量/其他指标
来源：数据来源或触发事件
```

### 类型标签

- `[公众号]` - 公众号运营相关
- `[小红书话术]` - 小红书回复话术
- `[关键词效果]` - 关键词采集效果
- `[审批案例-通过]` - 审批通过的案例
- `[审批案例-拒绝]` - 审批拒绝的案例
- `[策略优化]` - 策略优化建议

## 五、自动召回机制

### 工作原理

```
用户发起对话
    ↓
Mem0 自动分析上下文
    ↓
向量搜索相关记忆（topK=5）
    ↓
注入到对话上下文
    ↓
Echo 基于记忆做出决策
```

### 配置参数

- `autoRecall: true` - 自动召回
- `autoCapture: true` - 自动捕获
- `topK: 5` - 每次召回 5 条最相关记忆
- `searchThreshold: 0.5` - 相似度阈值

## 七、最佳实践

### 1. 及时存储
每次完成重要任务后，立即存储经验到 Mem0

### 2. 结构化格式
使用统一的格式，便于后续检索

### 3. 量化效果
尽可能记录数据指标（转化率、阅读量等）

### 4. 持续优化
定期（每周）分析记忆，提取优化建议

### 5. 避免重复
存储前先搜索，避免重复记录相同经验
