### Tab 2: 平台政策

- **显示字段：** 平台或主题、变化摘要、影响、是否立即影响、是否适合公众号、采集时间

### Tab 3: 高风险预警

- **排序：** 采集时间降序
- **高亮：** 风险等级为"高"的行标红
- **显示字段：** 全部字段

### Tab 4: 行业动态
- **筛选条件：** 分类 = "行业动态"
- **排序：** 采集时间降序
- **显示字段：** 平台或主题、变化摘要、是否适合公众号、来源链接、采集时间

## 使用流程

### xiaodong_crossborder_scout 写入流程

1. **查询字段结构**
```typescript
await feishu_bitable_app_table_field({
  app_token: "Z3DEboviLaJAlXsdUPvcItgrnZd",
  table_id: "tblmVbmO8MnY9UzE"
});
```

2. **判定是否需要写入**
- 高风险？→ 写入
- 立即影响生意？→ 写入
- 否 → 仅存 Mem0

3. **创建记录**
```typescript
await feishu_bitable_app_table_record({
  app_token: "Z3DEboviLaJAlXsdUPvcItgrnZd",
  table_id: "tblmVbmO8MnY9UzE",
  fields: {
    "分类": "高风险预警",
    "平台或主题": "法国公司注册",
    "变化摘要": "审核时效从 30 天延长至 45 天，材料要求增加银行流水证明",
    "对中国卖家的影响": "正在交付的订单可能延期 2 周，需提前告知客户",
    "风险等级": "高",
    "是否立即影响": "是",
    "是否适合公众号": "否",
    "来源链接1": {
      link: "https://...",
      text: "来源1"
    },
    "来源链接2": {
      link: "https://...",
      text: "来源2"
    },
    "验证状态": "已验证",
    "采集时间": Date.now(),
    "采集人": "xiaodong_crossborder_scout"
  }
});
```

- 通过 `sessions_send` 发送通知到 xiaodong 的会话
- 包含：风险类型、影响范围、建议动作

```javascript

  agent: "xiaodong",
  content: `🚨 跨境风险预警

**风险类型：** 高风险预警
**平台/主题：** 法国公司注册
**变化摘要：** 审核时效从 30 天延长至 45 天
**影响：** 正在交付的订单可能延期 2 周

📊 详情：https://99love.feishu.cn/base/Z3DEboviLaJAlXsdUPvcItgrnZd?table=tblmVbmO8MnY9UzE`
});
```
