def _is_chart_image(self, element):

    bbox = element["bbox"]

### 3. 头部间距优化 ✅
**实施内容：**
- 添加`_adjust_header_spacing()`方法，调整头部logo间距
- 识别逻辑：
  - 识别顶部50px内的所有图像
  - 按X坐标排序
- 调整算法：
  - 计算理想间距（均匀分布）
  - 第一个logo不动，后续依次调整
  - 保持总宽度不变

**效果：**
- ✅ Logo间距均匀
- ✅ 头部布局协调
- ✅ 间距误差 < 1px

**代码实现：**
```python
def _adjust_header_spacing(self, elements):
    """调整头部区域的间距"""
    header_images = [e for e in elements
                     if e["type"] == "image"
                     and e["bbox"][1] < 50]
    
    if len(header_images) < 2:
        return elements
    
    # 按X坐标排序
    header_images.sort(key=lambda e: e["bbox"][0])
    
    # 计算理想间距
    total_width = header_images[-1]["bbox"][2] - header_images[0]["bbox"][0]
    images_width = sum(img["bbox"][2] - img["bbox"][0] for img in header_images)
    ideal_spacing = (total_width - images_width) / (len(header_images) - 1)
    
    # 调整位置
    current_x = header_images[0]["bbox"][2]
    for i, img in enumerate(header_images[1:], 1):
        new_x0 = current_x + ideal_spacing
        width = img["bbox"][2] - img["bbox"][0]
        img["bbox"] = (new_x0, img["bbox"][1], new_x0 + width, img["bbox"][3])
        current_x = new_x0 + width
    
    return elements
```
