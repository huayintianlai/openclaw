# 公众号文章发布指南

**适用场景**：当 Browser 自动化失效时，使用此手动发布流程

---

## 一、准备工作

### 1. 获取文章文件

Echo 会将生成的文章保存在以下位置：

```
instances/kentclaw/data/workspace/echo/articles/
├── {record_id}.md        # Markdown 原文
└── {record_id}.html      # 转换后的 HTML（可直接粘贴）
```

### 2. 确认审批状态

在飞书"内容日历表"中确认文章审批状态为"已批准"：

- 表格链接：https://99love.feishu.cn/base/Z3DEboviLaJAlXsdUPvcItgrnZd?table=tblIv3haHPEPAd3Y&view=vewGAQfmMX
- 找到对应文章记录
- 确认"审批状态"字段为"已批准"

---

## 二、发布步骤

### Step 1：打开公众号后台

1. 访问：https://mp.weixin.qq.com
2. 使用微信扫码登录
3. 进入"素材管理" → "新建图文"

### Step 2：复制文章内容

**方案 A：使用转换后的 HTML（推荐）**

1. 打开 `{record_id}.html` 文件
2. 全选复制（Cmd+A, Cmd+C）
3. 粘贴到公众号编辑器（Cmd+V）
4. 检查格式是否正确

**方案 B：使用在线转换工具**

1. 打开 `{record_id}.md` 文件
2. 访问：https://md.openwrite.cn/ 或 https://lab.lyric.im/wxformat/
3. 粘贴 Markdown 内容
4. 点击"复制"按钮
5. 粘贴到公众号编辑器

### Step 3：调整格式

检查以下格式是否正确：

- ✅ 标题层级（一级标题、二级标题）
- ✅ 段落间距
- ✅ 列表格式（有序列表、无序列表）
- ✅ 加粗和引用
- ✅ 链接（如有）

### Step 4：插入图片（可选）

如果文章需要配图：

1. 点击编辑器中的"图片"按钮
2. 上传本地图片或从图片库选择
3. 调整图片位置和大小

### Step 5：预览检查

1. 点击"预览"按钮
2. 使用手机微信扫码查看
3. 检查手机端显示效果
4. 确认无误后返回编辑器

### Step 6：保存草稿或发布

**保存草稿**：
- 点击"保存"按钮
- 草稿会保存在"草稿箱"中
- 可以稍后继续编辑

**直接发布**：
- 点击"发表"按钮
- 选择"群发"或"定时发送"
- 确认发布

---

## 三、发布后操作

### 1. 更新飞书表格

在"内容日历表"中更新以下字段：

- **发布时间**：填写实际发布时间
- **公众号链接**：复制文章链接（格式：`https://mp.weixin.qq.com/s/xxxxx`）
- **审批状态**：保持"已批准"或改为"已发布"（如有此选项）

### 2. 记录到 Mem0（由 Echo 自动完成）

Echo 会自动将发布信息存储到 Mem0：

```
[公众号] ${文章标题} - 发布记录
时间：${发布日期}
链接：${公众号链接}
选题方向：${选题方向}
素材来源：${素材来源}
```

### 3. 定期更新阅读量

建议每周更新一次阅读量和转化客户数：

- 登录公众号后台查看阅读量
- 在飞书表格中更新"阅读量"字段
- 根据客户咨询情况更新"转化客户数"字段

## 四、常见问题

### Q1：粘贴后格式错乱怎么办？

**解决方案**：
1. 清空编辑器内容
2. 使用"纯文本粘贴"（Cmd+Shift+V）
3. 或使用在线转换工具重新转换

### Q2：图片无法显示怎么办？

**解决方案**：
1. 检查图片格式（支持 JPG、PNG、GIF）
2. 检查图片大小（单张不超过 10MB）
3. 重新上传图片

### Q3：链接无法点击怎么办？

**解决方案**：
1. 公众号文章中的外部链接需要设置"阅读原文"
2. 内部链接可以使用"超链接"功能
3. 或将链接文本化，让读者复制

### Q4：发布后发现错误怎么办？

**解决方案**：
1. 如果是小错误（标点、错别字），可以在后台"编辑"
2. 如果是重大错误，建议删除重发
3. 在飞书表格中记录错误原因，避免重复

## 五、优化建议

### 1. 提前准备素材

- 提前准备好配图（封面图、内文图）
- 提前准备好相关链接
- 提前确定发布时间

### 2. 保持发布节奏

- 建议每周固定时间发布（如周二、周四）
- 保持稳定频率，培养读者习惯

### 3. 数据分析

- 定期分析阅读量和转化率
- 识别高转化率的选题方向
- 优化内容策略
