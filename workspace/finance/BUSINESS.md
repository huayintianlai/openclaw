# 星光跨境 — 业务背景

## 公司概况

星光跨境是一家跨境电商服务公司，主营业务为欧洲公司代注册、电商平台代注册及海外电商平台现店销售。

## 合伙人结构

- **张建东** & **邓竣峰**，50/50 出资、50/50 分利
- 当前无销售员工，两位合伙人直接运营

## 利润分配与结算规则

### 收入端（毛利即时分配）

每笔订单完成后，**当天**按以下逻辑分配：

```
毛利 = 销售额 − 采购成本
每人分得 = 毛利 / 2
```

退款订单：公司利润归零，不参与分配。

### 成本端（月度均摊）

- 日常成本由两人各自垫付
- 每月 **10 号** 为结算日，计算当月垫付差额并均摊

```
差额 = |张建东垫付总额 − 邓竣峰垫付总额|
少付方向多付方转账 = 差额 / 2
```

## 业务线定义

### 1. 代注册服务

帮客户代注册欧洲公司及电商平台账号，收取服务费。利润 = 服务费 − 注册成本。

### 2. 现店业务

销售已注册完成的海外电商平台账号（现店），分为两种模式：

| 模式 | 说明 | 利润率特点 |
|---|---|---|
| **中介服务** | 为客户匹配现有店铺资源 | 利润率较高（无注册成本） |
| **自注册店铺销售** | 自行注册店铺后销售 | 利润率较低（需承担注册成本） |

### 3. 渠道合作（自注册店铺的供应链）

自注册店铺需要法人配合，法人配合成本 ¥5,000/人。目前有两个渠道：

#### 渠道胡 — 投资模式

- 渠道垫资法人费用 ¥5,000
- 我方承担公司注册成本 ¥10,000/公司
- 利润分成：我方 55%，渠道 45%
- 分配顺序：
  1. **回本期**：我方先收回 ¥10,000
  2. **渠道回本**：渠道收回 ¥5,000
  3. **分润期**：剩余净利润按 55/45 分配
- 净利润计算需扣除所有成本（注册成本、代注册成本等）

#### 渠道葛 — 买断制

- 我方垫付法人费 ¥5,000 + 公司注册费 ¥10,000
- 利润 100% 归我方

## 覆盖平台

Worten、Coupang、Cdiscount、Bol、乐华梅兰（Leroy Merlin）、Fruugo 等欧洲电商平台。

## 管理关注点

- 各渠道投资回报率（ROI）
- 未结算股东利润
- 月度成本控制与均摊准确性
- 退款订单对利润的影响
- 固定资产折旧对成本的影响
