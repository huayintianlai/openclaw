### 员工分红计算规则

```
IF 已退款 → ¥50（固定）
ELSE IF 公司利润 < 100 且 > 0 → 公司利润 × 30%
ELSE IF 无贡献 → ¥0
ELSE IF 客户复购 → (销售额 × 0.7% + 50) / 2
ELSE（默认/新客户）→ 销售额 × 0.7% + 50
```

## 核心 KPI 公式（用实际字段名）

### 收入侧

```
月度总收入     = SUM(销售额) WHERE 账期 = 目标月份 AND 员工贡献 ≠ "已退款"
月度总成本     = SUM(采购成本) WHERE 账期 = 目标月份
月度毛利       = 月度总收入 − 月度总成本
月度毛利率     = 月度毛利 / 月度总收入
```

### 分配侧

```
月度员工分红   = SUM(员工分红) WHERE 账期 = 目标月份
月度股东分红   = SUM(股东实际分红(单人)) WHERE 账期 = 目标月份
```

### 支出侧

```
月度总支出     = SUM(金额) WHERE 账期 = 目标月份
  ├─ 服务成本  = SUM(金额) WHERE 费用分类 = "服务成本"
  ├─ 管理费用  = SUM(金额) WHERE 费用分类 = "管理费用"
  ├─ 开发费用  = SUM(金额) WHERE 费用分类 = "开发费用"
  ├─ 固定资产  = SUM(金额) WHERE 费用分类 = "固定资产"
  └─ 投资      = SUM(金额) WHERE 费用分类 = "投资"
```

### 综合

```
月度净利润     = 月度毛利 − 月度总支出（扣除固定资产和投资类）
经营性现金流   = 月度毛利 − 月度总支出（仅经营性：服务成本 + 管理费用）
```

## 数据校验规则

1. **退款排除**：`员工贡献 = "已退款"` 的记录，公司利润为 0，不计入收入统计
2. **金额非零**：`销售额` 和 `采购成本` 必须 > 0（退款记录除外）
3. **日期完整**：所有记录必须有日期/付款时间
4. **枚举合规**：单选字段值必须在上述枚举范围内

## 披露要求

每次输出财务分析结果时，必须包含：

1. 数据时间范围（起止月份）
2. 口径声明：管理口径（非会计准则）
3. 退款订单数量及其对利润的影响
4. 未结算股东利润提示
5. 不确定项（如缺失数据、异常值）
