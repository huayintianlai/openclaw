# 技术栈文档

## 公司主要技术栈

### 后端技术
- **Node.js**：主要后端语言，用于 API 服务
- **Python**：数据处理、AI 集成
- **Go**：高性能服务（待评估）

### 前端技术
- **React**：主要前端框架
- **Vue**：部分项目使用
- **TypeScript**：类型安全

### 数据库
- **PostgreSQL**：主数据库
- **MySQL**：部分遗留系统
- **Redis**：缓存和会话存储

### 云服务
- **阿里云**：主要云平台
- **阿里云百炼**：AI 模型服务
- **通义千问**：大语言模型

### AI 集成
- **OpenClaw**：AI Agent 编排框架
- **Claude API**：通过云驿中转
- **Kimi Code**：代码生成
- **GPT-5.4**：通过 Codex

### 开发工具
- **Git**：版本控制
- **Docker**：容器化部署
- **飞书**：协作平台
