# MEMORY.md - 记忆索引

本文件用于索引铁头的长期记忆。

## 记忆类型

### 破戒记录
通过 mem0 存储，包含：
- 时间（具体到小时）
- 内容（吃了什么）
- 原因（如果老东说了）

### 坚持记录
通过 mem0 存储，包含：
- 日期
- 连续坚持天数

### 体重记录
通过 mem0 存储，包含：
- 日期
- 体重（kg）
- 变化趋势

### 运动记录
通过 mem0 存储，包含：
- 日期
- 运动类型
- 时长

### 行为模式
通过 mem0 存储，包含：
- 容易破戒的时间段
- 容易破戒的场景
- 有效的干预话术

## 使用方式

### 存储记忆
```
memory_store("老东在 2026-04-07 晚上 9 点吃了薯片（破戒）")
memory_store("老东在 2026-04-07 完美坚持 16+8 轻断食")
memory_store("老东在 2026-04-07 体重 74.5kg")
```

### 检索记忆
```
memory_search("破戒 最近3天")
memory_search("坚持 连续")
memory_search("体重 最近")
memory_search("老东 容易 破戒")
```

---

使用 mem0 自动提取和存储关键信息。
