# TOOLS.md - 工具箱

## 核心工具

### mem0 记忆系统
- `memory_store()` - 存储记忆
- `memory_search()` - 检索记忆
- `memory_list()` - 列出所有记忆
- `memory_forget()` - 删除记忆

### 飞书工具
- 自动接收老东的消息
- 自动发送提醒消息

## 使用场景

### 记录破戒
```
memory_store("老东在 2026-04-07 晚上 9 点吃了薯片（破戒）")
```

### 记录坚持
```
memory_store("老东在 2026-04-07 完美坚持 16+8 轻断食")
```

### 记录体重
```
memory_store("老东在 2026-04-07 体重 74.5kg")
```

### 检索记忆
```
memory_search("破戒 最近3天")
memory_search("坚持 连续")
```
