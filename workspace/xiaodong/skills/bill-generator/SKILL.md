# PDF账单生成器

## Description

根据身份证照片和PDF模板，自动生成新的账单文件。支持智能字段识别、AI值生成，保持原PDF样式完全一致。

## When to use

- 用户需要生成电费、水费、燃气费等账单
- 用户提供了身份证照片或身份信息
- 用户想要修改现有PDF账单的内容

## Tools

- `bill_ocr_id_card` - 识别身份证信息
- `bill_generate` - 生成账单PDF

## Procedure

### 步骤1：获取身份信息

如果用户提供了身份证照片：
1. 使用 `bill_ocr_id_card` 工具识别
2. 展示识别结果（姓名、身份证号、地址）
3. 请用户确认信息是否正确
4. 如果有误，让用户手动提供正确信息

如果用户直接提供身份信息：
1. 收集姓名（必需）
2. 收集地址（可选）
3. 收集身份证号（可选）
4. 收集电话（可选）

### 步骤2：获取PDF模板

1. 询问用户PDF模板文件路径
2. 验证文件是否存在
3. 如果文件不存在，请用户提供正确路径

### 步骤3：收集自定义字段（可选）

询问用户是否需要自定义某些字段：
- 金额（如果不提供，AI会自动生成合理值）
- 账期（如果不提供，使用当前月份）
- 日期（如果不提供，使用当前日期）
- 其他字段

### 步骤4：生成账单

1. 调用 `bill_generate` 工具
2. 传入：
   - template_path: PDF模板路径
   - identity: 身份信息对象
   - custom_fields: 自定义字段（如果有）

### 步骤5：展示结果

1. 告知用户生成成功
2. 提供输出PDF文件路径
3. 提供对比图路径（供用户预览）
4. 列出所有修改的字段
5. 询问用户是否满意

### 步骤6：后续处理

如果用户不满意：
- 询问需要调整哪些字段
- 使用新的custom_fields重新生成

如果用户满意：
- 确认完成
- 提示用户可以打开文件查看

## Notes

- 生成的PDF会保持原模板的所有样式（字体、字号、颜色、位置）
- AI生成的字段（标记为⚡）会遵循原有规律
- 对比图可以帮助用户快速检查修改效果
- 支持多次调整，直到用户满意
- 所有生成的文件保存在 workspace/xiaodong/bills/ 目录

## Error Handling

- 如果OCR识别失败，提示用户手动输入信息
- 如果模板文件不存在，请用户提供正确路径
- 如果生成失败，展示错误信息并询问是否重试
- 如果字段识别不准确，可以通过custom_fields手动指定
